    no | false | yes | true

    yes

    dialog-height

    string

    List picker dialog height

    dialog-width

    string

    filter

    no | false | yes | true

    Indicates whether or not to show values filtering input

    yes

    number

    Number of records by page. Initial number of registries in queries when using pagination in server side

    10

    text-input-enabled

    no | false | yes | true

    Indicates whether or not to allow the user to edit the list picker manually

    yes

    string

    Visible columns. Separated by ‘;’
